Installatie

1. Downloaden GIT

Git is een versiebeheersysteem, dat wij zullen gebruiken om de User Interface en bijbebehorende extensies te downloaden en te updaten.Tijdens de Git installatie kan je alles standaard laten zoals het staat. Je zou de master-branch kunnen veranderen van naam naar main, deze naamgeving word vaker gebruikt dan master, maar speelt in deze handleiding geen rol.

2. Downloaden Python 3.10.6

Python is een programmeertaal dat veel gebruikt word bij onder ander het maken van A.I. applicaties, waarond Stable Diffusion. Hiervoor downloaden we het dus ook.Vergeet niet tijdens de Python Installatie (die je onderaan de pagina kan vinden)om 'Add python.exe to PATH' aan te vinken, dit maakt het voor andere applicaties makkelijker om Python te vinden. Nieuwere versies zoals 3.12 zouden problemen kunnen hebben met Stable Diffusion te runnen, daarom downloaden we voor de zekerheid 3.10.

3. Kopieëren stable-diffusion-webui Github link

We gaan iets downloaden genaamd stable-diffusion-webui, dit is de User Interface waarmee je kan interacteren met de generatie modellen.Klik op de link hierboven en klik vervolgens op de groene '<> Code' knop, kopieer vervolgens de HTTPS link op het scherm (kijk screenshot).

4. webui klonen

Navigeer naar een lege folder waar je stable-diffusion-webui wilt downloaden, type vervolgens in de adresbalk boven aan `cmd`, dit opent commandprompt. Type in deze commandprompt `git clone [DE HTTPS LINK]` (Kijk screenshot), dit download de webui op je apparaat met de bijbehorende folders en code. Druk op 'Enter' en de download zal beginnen, dit kan even duren.

5. Batch bestand aanpassen

Wacht totdat de download klaar is en navigeer vervolgens in de folder naar het bestand genaamd `webui-user.bat`, dit is het bestand dat ervoor zorgt dat je de webui kan gebruiken. Klik met de rechtermuisknop op dit bestand en klik vervolgends op `edit`/`bewerken`, voeg vervolgens de volgende 2 dingen toe:1. Ga naar de locatie waar je Python hebt gedownload, dit kan je doen door het met de windowstoets/commandtoets hiernaar te zoeken en vervolgens het pad uit de adresbalk to kopieëren. Plak dit vervolgens achter `set PYTHON=` zoals te zien is in het voorbeeld hieronder.2. Voeg voor `call webui.bat` de zin `git pull` toe. Dit zorgt ervoor dat wanneer je Stable Diffusion wilt gebruiken, er automatisch geupdate word naar wat nieuw is.

6. Downloaden modellen

Het zou zo kunnen zijn dat in de folder 'models/Stable-Diffusion', het model SD v1.5 al gedownload is. Dit is een oud maar populair Base Model waarvan veel andere modellen getraind zijn. Momenteel zijn er modernere en beter getrainde modellen beschikbaar.Volg deze link naar CivitAi en download hier vervolgens de nieuwste versie van het Dreamshaper model. Dit is een populair model dat actief word bijgehouden met vernieuwde versies en is in het algemeen beter getraind dan SD1.5.

7. Opstarten stable-diffusion-webui

Start de webui door webui-user.bat' op te starten, dit kan even duren aangezien er veel extra gedownload moet worden. Zodra 'webui-user.bat' runt kan je in je browser met het webadres http://127.0.0.1:7860/ gebruik maken van Stable Diffusion, dit opent al uit zichzelf na de download.
